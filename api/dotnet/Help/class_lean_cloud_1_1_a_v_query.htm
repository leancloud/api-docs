Detailed Description

The AVQuery class defines a query that is used to fetch AVObjects. The most common use case is finding all objects that match a query through the FindAsync() method.

This sample code fetches all objects of class "MyClass":

AVQuery query = new AVQuery("MyClass");
IEnumerable<AVObject> result = await query.FindAsync();

A AVQuery can also be used to retrieve a single object whose id is known, through the GetAsync(string) method. For example, this sample code fetches an object of class "MyClass" and id myId.

AVQuery query = new AVQuery("MyClass");
AVObject result = await query.GetAsync(myId);

A AVQuery can also be used to count the number of objects that match the query without retrieving all of those objects. For example, this sample code counts the number of objects of the class "MyClass".

AVQuery query = new AVQuery("MyClass");
int count = await query.CountAsync();

Detailed Description

In the event that you would like to use the LeanCloud SDK from a completely portable project, with no platform-specific library required, to get full access to all of our features available on LeanCloud.com (A/B testing, slow queries, etc.), you must set the values of this struct to be appropriate for your platform.

Any values set here will overwrite those that are automatically configured by any platform-specific migration library your app includes.

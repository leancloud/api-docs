enum LeanCloud.Realtime.AVRealtime.OfflineMessageStrategy
strong

登录时的离线消息下发策略

Enumerator
Default 

服务器将所有离线消息一次性在登录之后马上下发下来

UnreadNotice 

不再下发未读消息，而是下发对话的未读通知，告知客户端有哪些对话处于未读状态

UnreadAck 

ack 和 read 分离, ack 不会清理未读消息

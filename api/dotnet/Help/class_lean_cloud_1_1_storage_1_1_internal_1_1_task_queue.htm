Enqueues a task created by taskStart . If the task is cancellable (or should be able to be cancelled while it is waiting in the queue), pass a cancellationToken.

Template Parameters
TThe type of task.
Parameters
taskStartA function given a task to await once state is snapshotted (e.g. after capturing session tokens at the time of the save call). Awaiting this task will wait for the created task's turn in the queue.
cancellationTokenA cancellation token that can be used to cancel waiting in the queue.
Returns
The task created by the taskStart function.

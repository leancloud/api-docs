inline

向该对话发送消息。

Parameters
avMessage消息体
receipt是否需要送达回执
transient是否是暂态消息，暂态消息不返回送达回执(ack)，不保留离线消息，不触发离线推送
priority消息等级，默认是1，可选值还有 2 ，3
will标记该消息是否为下线通知消息
pushData如果消息的接收者已经下线了，这个字段的内容就会被离线推送到接收者

例如，一张图片消息的离线消息内容可以类似于：[您收到一条图片消息，点击查看] 这样的推送内容，参照微信的做法
